# forkify Project

Recipe application with custom recipe uploads.
Also added some extra features such as:
1. Showing the nutritions data of the recipe that you choose.
2. Showing the amount of pages in a results bar.
3. Ingridient validation. Can add more than a six ingridients.
4. Warning messages, how to fill input ingridients fields.
5. Also can delete ingridients, if you have spelt it uncorrectly or wanna change something.
